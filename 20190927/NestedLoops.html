Richard Fenix

Problem Link: https://programmingbydoing.com/a/basic-nested-loops.html

Solution:

public class BasicNestedLoops {

    public static void main( String[] args )

    {

        for ( int a=1; a <= 5; a++ )

        {

            for ( int b=1; b <= 5; b++ )

            {

                System.out.print( "(" + a + "," + b + ") " );

            }

            System.out.println();

        }

    }

}

Output:

(1,1) (1,2) (1,3) (1,4) (1,5)

(2,1) (2,2) (2,3) (2,4) (2,5)

(3,1) (3,2) (3,3) (3,4) (3,5)

(4,1) (4,2) (4,3) (4,4) (4,5)

(5,1) (5,2) (5,3) (5,4) (5,5)
