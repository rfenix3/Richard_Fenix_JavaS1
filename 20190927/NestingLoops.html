Problem Link: https://programmingbydoing.com/a/nesting-loops.html

  1. Nesting Loops:
  1. Look at the first set of nested loops ("CN"). Which variable changes faster? Is it the variable controlled by the outer loop (c) or the variable controlled by the inner loop (n)?

The variable controlled by the inner loop changes faster as it needed to complete the loop for every outer loop variable (A, B, C, D, and E).

  1. Change the order of the loops so that the "c" loop is on the inside and the "n" loop is on the outside. How does the output change?

The c loop will change faster since it is in the inner loop this time. The output will be:

1 A

1 B

1 C

1 D

1 E

2 A

2 B

...

  1. Look at the second set of nested loops ("AB"). Change the print() statement to println(). How does the output change? (Then change it back to print().)

                The print output becomes per line instead of 1 long line. New output is:

1-1

1-2

1-3

2-1

2-2

2-3

3-1

...

  1. Add a System.out.println() statement after the close brace of the inner loop (the "b" loop), but still inside the outer loop. How does the output change?

There is now a line break for each outer loop parameter value.

(a=1 then line break, a=2 then line break...)

Output became:

1-1 1-2 1-3

2-1 2-2 2-3

3-1 3-2 3-3
